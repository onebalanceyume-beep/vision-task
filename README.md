## 🚀 初回セットアップ手順

### Step 1: ファイルをGitHubに上げ替える

1. 既存の `vision-task` リポジトリを開く
2. 古いファイル（index.html等）を **この新しいものに置き換え**
3. commit

### Step 2: Googleスプレッドシート作成（ヨシさん作業）

1. Googleドライブで新規スプレッドシート作成
2. タイトル: **`VISION_TaskApp_DB`**
3. 下部のシートタブに **4つのシート** を作成：
   - `tasks`
   - `projects`
   - `members`
   - `referrers`
4. ヘッダー行は空のままでOK（GASが自動で書きます）
5. スプレッドシートのURLを記録（後で使わないけど念のため）

### Step 3: GASコード貼り付け

1. スプレッドシート画面で **「拡張機能」→「Apps Script」**
2. デフォルトコード `function myFunction()...` を削除
3. アプリのメニュー →「📋 GASコードをコピー」→「コピー」ボタン
4. Apps Scriptに貼り付け
5. **保存**（💾アイコン or Ctrl+S）

### Step 4: デプロイ（公開）

1. Apps Script画面で右上の **「デプロイ」→「新しいデプロイ」**
2. 歯車アイコン → **「ウェブアプリ」** を選択
3. 設定：
   - 説明: `VISION app v1`（任意）
   - 次のユーザーとして実行: **自分**
   - アクセスできるユーザー: **全員** ← **これ重要！**
4. **「デプロイ」** クリック
5. 初回のみ承認画面が出る → 「アクセスを承認」→ 自分のGoogleアカウント選択 → 「Advanced」→ 「安全でないページに進む」→ 「許可」
6. 発行された **ウェブアプリのURL** をコピー（`https://script.google.com/macros/s/...../exec`）

### Step 5: アプリに設定

1. アプリのメニュー →「☁️ クラウド同期設定」
2. 「GAS WebアプリURL」に貼り付け
3. **「保存」**
4. **「接続テスト」** で確認
5. **「⬆️ この端末 → クラウド」** をクリックして初回アップロード

## 🔄 同期の仕組み

- **保存時に自動でクラウドに送信**（5秒デバウンス）
- **手動で取得**: 上部の ⟳ ボタン
- **オフライン時も動作**（次回オンライン時に同期）
- **複数端末で同時編集した場合は後勝ち**（最後に保存した方が残る）

## 💡 使い方のコツ

### 紹介者機能の活用例
- **ケース1**: 田中様＋佐藤様の2名が紹介 → タスクに両方チェック
- **ケース2**: 紹介者ビューで「田中様は美容系で強い」「佐藤様はリフォーム強い」が可視化

- **遠くから全体を見る**: ⌂ボタンでリセット

- **一覧と詳細の切替**: ミニマップの任意の位置をタップで瞬間移動

### クラウド同期活用例
- **PC**: 朝にタスク全体を設計
- **iPhone**: 現場でサクッと完了チェック
- **スタッフ**: 自分の担当タスクを確認・完了

## ⚠️ 注意点

- **GASの無料枠**: 1日あたりスクリプト実行6時間まで（個人利用では絶対超えない）
- **同期タイミング**: 即時ではなく5秒後（連続入力のデータ損失防止）
- **写真**: 同期時も含まれる（大量に貼ると同期が遅くなる）
- **削除は慎重に**: 他端末で同期された後に消すと復元できない

## 🆘 トラブルシューティング

### 同期しない
- アプリメニュー → 「接続テスト」確認
- GASの「アクセスできるユーザー: 全員」になってるか確認
- URLに `/exec` が付いてるか確認

### 画面が古いまま
- アプリメニュー → 「キャッシュ削除＆再読込」

### データが消えた
- 他端末で「クラウド→この端末」を実行すれば復元可能
- 週1でメニュー →「データ書き出し」でバックアップ推奨
