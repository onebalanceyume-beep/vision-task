# VISION タスク管理アプリ 使い方

## 🎯 コンセプト

**すべてのタスクは「ビジョン達成」のためにある**

- 画面最上部に**ビジョン**が常時表示されます
- 「心と体を元気に。カッコよく素敵に。／年商10億・利益2億」
- これを毎日見ることで判断軸がブレません

## 📱 4つのビュー

### 🗺 マップ（マインドマップ）
- ビジョン → プロジェクト → タスクの3階層ツリー
- タスクをドラッグで自由配置
- ピンチでズーム／スワイプで移動
- タスク同士を矢印でつなぐ（依存関係）

### 📋 ボード（かんばん）
- 未着手／進行中／完了 の3列表示
- カードタップで編集
- ピン留めタスクが上部に

### 👥 担当者
- 担当者を選ぶと、その人が全プロジェクト横断で担当してるタスク一覧
- 🔴期限切れ / ⭐今週やる / 🟡進行中 / ⚪未着手 / ✅完了 で自動グルーピング
- 「未割当」タスクも一覧可能（次の人に任せやすい）

### ⭐ 今週
- 🔴期限切れ（要注意タスク）
- ⭐ピン留め（今週やる）
- 📅今週期限
- 🚗今週の訪問予定
- 朝これを見るだけで1日の動きが決まる

## 🆕 タスクに登録できる情報

- タスク名・メモ（なぜやるか）
- **KPI・数値目標**（例：契約3件、売上100万）
- **担当者**（複数OK）
- ステータス（未着手／進行中／完了）
- **ビジョン貢献度**（売上直結／ブランディング／仕組み化）
- ★**今週やるピン留め**
- 色（8色から）
- 期日／**訪問予定日**
- **紹介元**メモ
- **写真添付**（自動圧縮）
- **完了履歴ログ**（誰がいつ完了したか記録）

## 📌 主な操作

### タスク追加
- 左下の **＋** ボタン

### タスク編集
- タスクをタップ

### タスク移動（マインドマップ）
- 長押しドラッグ

### タスク同士をつなぐ（依存関係）
1. タスクをタップ → 編集画面
2. 「↗ 他のタスクとつなぐ」
3. 相手のタスクをタップ

### プロジェクト追加
- 上部タブの「＋」または メニュー → プロジェクト管理

### 担当者追加
- メニュー → 担当者管理
- または 担当者ビューの「＋担当者追加」

### ビジョン変更
- 画面上部「✎」ボタン

## 💾 データ

- **保存**: ブラウザ内（localStorage）
- **バックアップ**: メニュー → データ書き出し
- **復元**: メニュー → データ読み込み
- **端末が故障したら消えます**→週1のバックアップ推奨

## 📲 インストール（iPhone）

1. このアプリをSafariで開く（ホスティング必要）
2. 共有ボタン → 「ホーム画面に追加」
3. ロゴ付きアプリとして起動

## 🔧 画面が更新されない時

メニュー → 「キャッシュ削除＆再読込」

## 💡 プロとしてのオススメ使い方

### 朝のルーティン（3分）
1. **今週**ビューを開く
2. 🔴期限切れをゼロに（今日中に片付ける or 延期設定）
3. ⭐ピン留めから「今日の3つ」を決める

### 週次ルーティン（10分）
1. 各プロジェクトで進捗確認
2. 次週やるタスクにピン留め
3. 担当者ビューで偏り確認 → 再分配

### 月次ルーティン（15分）
1. 完了タスクを振り返り（ログ確認）
2. ビジョン貢献度別に集計（売上直結が少ないなら要対策）
3. データバックアップ
